# Multigrid_learning
TensorFlow code for Learning Generative ConvNets via Multi-grid Modeling and Sampling
## Prerequisites
Tensorflow (>= 1.2). Plase refer to the tensorflow [installing instruction](https://www.tensorflow.org/install/). <br>
Python packages: skimage pprint scipy sklearn
## Run the code
(1) Copy the training data into ./data/dataset_name <br>
(2) Run the code 
```
python main.py --dataset_name dataset_name 
```
